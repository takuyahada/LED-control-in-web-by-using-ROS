LED control in web by using ROS

## Demonstration
https://www.youtube.com/watch?v=IhS-2Nt7BMk&feature=youtu.be

## Requirements
* Raspberry Pi 3 Model B
 * ubuntu

* 3 LEDs
 * connected with GPIO 15 , 16 and 18 pins
 
## Usage
* Push LED1 or LED2 or LED 3 or LED OFF in the web.
